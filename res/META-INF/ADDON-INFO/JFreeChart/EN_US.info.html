JFreeChart - a free Java chart library that makes it easy to display professional quality charts in Java applications.

It is provided by JFree.org - site of free software projects targetting the Java platform, owned and operated by David Gilbert and Thomas Morgner.

This library also uses following library:

JCommon - library that contains the common classes, which provide some global utility functionality for both GUI and non-GUI applications; provided by JFree.org - site of free software projects targetting the Java platform, owned and operated by David Gilbert and Thomas Morgner.
