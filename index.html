DOWNLOAD / INSTALL

If you’re running Ubuntu, it’s recommended to install Bias via PPA-repository using the following command:

sudo add-apt-repository ppa:ki0n/software && sudo apt-get update && sudo apt-get install bias

Otherwise, use one of the download options below.

→ DEB (Debian based Linux distros, e.g. Ubuntu)
→ DMG (Mac OS X)
→ APP.ZIP (Mac OS X)
→ ZIP (Cross-Platform: Linux, Mac OS X, Windows, etc.)


NOTE: You must have Java 11+ installed to run Bias.
If you don't have it installed yet, you can download it from here.
(Installing via DEB package will resolve dependencies automatically though ;))
